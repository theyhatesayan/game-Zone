# 🎮 Game Zone

Game Zone ek interactive web-based gaming platform hai jahan users simple aur fun browser games play kar sakte hain. Yeh project frontend skills ko showcase karta hai using modern UI design, smooth animations aur responsive layout.

## 🚀 Features

* 🎯 Multiple mini games (add your game names here)
* ⚡ Fast and smooth gameplay
* 🎨 Modern UI with animations
* 📱 Fully responsive (mobile + desktop)
* 🖱️ Interactive buttons and effects

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Project Structure

```
Game-Zone/
│── index.html
│── style.css
│── script.js
│── /assets
```

## 🎮 How to Play

1. Open the website
2. Choose your game
3. Start playing and enjoy 🎉

## 💡 Future Improvements

* Add more advanced games
